# context7

Context7 library docs.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y @upstash/context7-mcp
```

## Authentication

No authentication required.

## Tools

- resolve-library-id
- get-library-docs

## Capabilities

`code_docs` `library_docs` `api_reference` `programming`
