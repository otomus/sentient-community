# duckduckgo

DuckDuckGo search.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y duckduckgo-mcp-server
```

## Authentication

No authentication required.

## Tools

- search
- fetch_content

## Capabilities

`web_search` `knowledge` `facts` `news` `lookup`
