# airbnb

Airbnb listing search.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y @openbnb/mcp-server-airbnb
```

## Authentication

No authentication required.

## Tools

- search_listings
- get_listing_details

## Capabilities

`travel` `airbnb` `accommodation` `vacation` `rental`
