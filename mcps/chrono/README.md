# chrono

Time/timezone utilities.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y @jmoak/chrono-mcp
```

## Authentication

No authentication required.

## Tools

- get_current_time
- convert_timezone
- add_time
- subtract_time
- diff_time
- format_time

## Capabilities

`time` `timezone` `date` `clock` `schedule`
