# rss_reader

RSS feed reader.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y rss-reader-mcp
```

## Authentication

No authentication required.

## Tools

- subscribe_feed
- list_feeds
- get_articles
- get_article_content

## Capabilities

`news` `rss` `feeds` `articles` `blog`
