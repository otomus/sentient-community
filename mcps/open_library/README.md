# open_library

Open Library book search.

## Installation

This MCP server is installed automatically by sentient-core. To use it manually:

```bash
npx -y mcp-open-library
```

## Authentication

No authentication required.

## Tools

- get_book_by_title
- get_authors_by_name
- get_author_info
- get_book_cover

## Capabilities

`books` `authors` `literature` `reading` `library`
