# Contributing to Sentient Community

Sentient's core manages nerve synthesis, qualification, and tool fabrication. Contributions to this repo are generated by the system — not manually authored.

## How Contributions Work

1. Sentient synthesizes and qualifies a nerve on your machine
2. The CLI exports the nerve's proven identity (description, prompt, examples, tools, test cases)
3. The CLI submits a PR to this repo

```bash
python cli.py contribute nerve my_nerve
```

That's it. The core handles bundle structure, schema compliance, and validation.

## What Gets Exported

A nerve bundle contains **identity only** — no runtime code:
- `bundle.json` — description, role, system prompt, examples, tool references, model adapter scores
- `test_cases.json` — the qualification test bank
- `tools/` — tool specifications and implementations (any language)

The nerve runtime (`nerve.py`) is **never exported** — it's always generated from the local template on import.

## Contribution Requirements

These are enforced automatically by the CLI and CI:

- **Qualification score >= 0.70** — the nerve must have passed qualification
- **At least 4 test cases** with at least 1 core and 1 negative test
- **No secrets** — no API keys, tokens, passwords, or private keys
- **No absolute paths** — no `/Users/`, `/home/`, `C:\Users\`
- **Safe tool code** — no dangerous calls (`eval`, `exec`, `subprocess` in Python; `child_process`, `eval` in JS)
- **Schema valid** — all JSON files must conform to schemas in `schemas/`

## Brain Adapters

Brain adapters are also exported by the CLI:

```bash
python cli.py export brain
```

This captures the system prompt, model config, and qualification scores for your current brain model.

## Connectors

Connector contributions (WhatsApp, Telegram, Discord, etc.) are full implementations submitted manually via PR. Use the connector PR template.

## CI Validation

All PRs are validated automatically:
- JSON schema validation against `schemas/`
- Secret scanning
- Tool safety checks
- Structural completeness
