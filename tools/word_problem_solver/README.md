# word_problem_solver

Solve mathematical problems including arithmetic, algebra, and word problems using an external solver API.

## Usage

```python
from tool import run

print(run("What is 25 * 4 + 10?"))
print(run("Solve for x: 2x + 5 = 15"))
```

## Parameters

- `query` (string, required): Mathematical problem or word problem to solve.
