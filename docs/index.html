What is Arqitect?

Arqitect is a community hub for the autonomous agent framework. It hosts modular components that let you build, extend, and share agent capabilities — from IoT control to document processing, from web browsing to smart home integration.

01
Discover

Browse the catalog of tools, nerves, connectors, and MCP integrations available for your agent.

02
Compose

Nerves combine system prompts, tools, and test cases into modular behaviors. Mix and match to build your agent's skill set.

03
Connect

Deploy connectors to bridge your agent to messaging platforms — Discord, Slack, Telegram, WhatsApp, and more.

04
Contribute

Submit new connectors, tools, and MCP server references via PR. All contributions are validated by CI.
